# VisionBoard AI – Logo Concept

## Core Concept
A human hand with the index finger extended, leaving a glowing cyan trail that forms
a stylised circuit-board node — representing the fusion of natural human gesture and
digital/AI intelligence.

## Symbol Elements

| Element | Meaning |
|---|---|
| Hand silhouette | Human gesture / natural interaction |
| Glowing fingertip | The drawing cursor — where intent becomes reality |
| Cyan neon trail | Data stream / AI processing |
| Circuit node at trail end | Technology backbone |
| Subtle grid background | Digital canvas / whiteboard |

## Typography
- Wordmark: **VisionBoard** in `Exo 2` Bold, tracked +20
- Sub-label: **AI** in `Exo 2` Light, cyan `#06B6D4`, same cap-height as wordmark
- Tagline: *Transform Gestures into Ideas.* in `Inter` Regular, 60% opacity white

## Icon Versions

### Full Logo (horizontal)
```
[Hand icon]  VisionBoard AI
             Transform Gestures into Ideas.
```

### Square App Icon
Hand silhouette on `#0F172A` background, fingertip glowing in `#06B6D4`,
trail curves into a stylised "V" for VisionBoard.

### Minimal Mark
Just the "VB" lettermark in `#06B6D4` on `#0F172A` — used in toolbar top-left.

## Color Usage in Logo

| Token | Hex | Usage |
|---|---|---|
| Background | `#0F172A` | Icon background, dark surfaces |
| Primary (Cyan) | `#06B6D4` | Hand glow, wordmark accent, borders |
| White | `#F1F5F9` | Main wordmark text |
| Dim | `#64748B` | Tagline text |

## Suggested Tools to Build Logo
- **Figma** — vector design with glow effects
- **Adobe Illustrator** — professional vector export
- **Inkscape** (free) — SVG export for GitHub README badge
